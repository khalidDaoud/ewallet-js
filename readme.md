# ewallet-js
This is js library for ewallet

# Installation
```
npm install ewallet-js
```

# Admin
```
Ewallet = require("ewallet-js");
ewallet = new Ewallet.Admin("http://localhost:4000/api");
```

# Client
```
Ewallet = require("ewallet-js");
ewallet = new Ewallet.Client("http://localhost:4000/api");
```
